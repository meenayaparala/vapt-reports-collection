# SQL Injection Reports

This folder contains detailed walkthroughs and vulnerability analysis for multiple **SQL Injection (SQLi)** labs. Each report documents the attack scenario, methodology, payloads used, and lessons learned.

### Lab Reports

1. **01-hidden-data-sqli** – Accessing hidden content using basic SQL injection.
2. **02-login-bypass-sqli** – Bypassing login authentication via SQLi.
3. **03-oracle-version-sqli** – Extracting Oracle database version.
4. **04-mysql-mssql-version-sqli** – Retrieving MySQL/MSSQL version info.
5. **05-non-oracle-db-content** – Extracting database content from non-Oracle DBs.
6. **06-oracle-db-content** – Reading Oracle-specific table content.
7. **07-union-find-columns** – Determining the number of columns via UNION.
8. **08-union-find-text-column** – Finding a column that can return string data.
9. **09-union-other-table-data** – Extracting data from a different table.
10. **10-union-retrieve-multiple-values** – Retrieving multiple values using UNION SELECT.

Each report follows a consistent format for easier understanding and quick review.
